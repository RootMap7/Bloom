# Bloom

A mobile app for couples to share experiences together.

## Getting Started

This project is built with Flutter for cross-platform mobile development.

### Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK
- Android Studio / Xcode (for iOS development on macOS)

### Installation

1. Install Flutter dependencies:
```bash
flutter pub get
```

2. Run the app:
```bash
flutter run
```

### Project Structure

```
lib/
  main.dart          # Entry point of the application
  screens/           # App screens
  widgets/           # Reusable widgets
  models/            # Data models
  services/          # Business logic and API services
  utils/             # Utility functions
```
